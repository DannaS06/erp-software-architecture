# Sistema ERP – Taller de Documentación de Arquitectura

## Descripción

Este repositorio contiene el desarrollo del taller de documentación de arquitectura de software aplicado a un **Sistema ERP**.  

El objetivo es modelar la arquitectura del sistema utilizando el modelo **C4**, diagramas UML y la plantilla **arc42**, aplicando buenas prácticas de documentación.

El ERP permite gestionar diferentes módulos empresariales como:

- Compras  
- Facturación  
- Stock y Costos  
- Activos Fijos  
- Empleados  
- EIS (Executive Information System)
